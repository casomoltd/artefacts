# Reference Calculations — 2026/27

Step-by-step workings for each test case using HMRC rules.

All income tax and NI rates are frozen from 2025/26. Only the
student loan Plan 2 threshold changes (£28,470 → £29,385).

## Rates

- Personal allowance: £12,570 (tapers £1 per £2 over £100k)
- Basic 20%: £0–£37,700 | Higher 40%: £37,700–£125,140
- NI: primary threshold £12,570, UEL £50,270, main 8%, upper 2%
- Student loan Plan 2 threshold: £29,385 @ 9%

## Case A — £45,000, 5% NET pay pension, Plan 2

Tax and NI unchanged from 2025/26. Only student loan differs.

| Step | Working | Value |
|---|---|---|
| Tax total | (same as 2025/26) | £6,036 |
| NI total | (same as 2025/26) | £2,594.40 |
| Student loan | (45,000 − 29,385) × 9% | £1,405.35 |
| **Net** | 45,000 − 2,250 − 6,036 − 2,594.40 − 1,405.35 | **£32,714.25** |
