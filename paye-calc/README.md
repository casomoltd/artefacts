# paye-calc Test Fixtures

Golden-value regression data for
[@casomoltd/paye-calc](https://github.com/casomoltd/paye-calc).

## Fixtures

| File | Description |
|---|---|
| `fixtures/annual-calculations.csv` | Annual take-home pay across all pension bases |
| `fixtures/nhs-pension-calculations.csv` | NHS pension employer-basis calculations (ref: paye-calc#12) |
| `fixtures/scottish-calculations.csv` | Scottish income tax band calculations |
| `fixtures/student-loan-calculations.csv` | Student loan plan deductions |
| `fixtures/hmrc-ni-crosscheck.csv` | HMRC NI Category A monthly/annual crosscheck |
| `fixtures/hmrc-tax-periods.csv` | HMRC cumulative and W1/M1 tax period test data |
